┌─────────────────────────────────────────────────────────────────────────┐
│ 🚨 TOP 10 PITFALLS (Learn to Recognize These!)                         │
└─────────────────────────────────────────────────────────────────────────┘

1. ❌ Forgetting ICC in cluster RCT
   → NSN becomes dramatically underestimated
   → Design Effect typically 1.5-2.5x

2. ❌ Confusing OR and RR
   → OR ≈ RR ONLY if disease is rare (<10%)
   → Different interpretations!

3. ❌ Adjusting on mediators
   → Removes the signal you're trying to measure
   → Only adjust on CONFOUNDERS

4. ❌ No temporal relationship in case-control
   → Can't claim causality
   → Temporal ambiguity is inherent

5. ❌ Chi-square with expected frequency < 5
   → Use Fisher Exact instead!
   → p-values invalid otherwise

6. ❌ Cox without testing proportional hazards
   → Core assumption of the model
   → Test interaction term with log(time)

7. ❌ Assuming multiplicative = additive interaction
   → They can completely disagree!
   → Must calculate RERI if interested

8. ❌ Ecological study → individual inference
   → Ecological fallacy!
   → Group-level ≠ individual-level

9. ❌ Complete case analysis without justification
   → Selection bias if RCT
   → Need to justify why acceptable

10. ❌ Non-inferiority in Per-Protocol
    → Should be ITT!
    → Doesn't protect the margin

TEMPLATE: NSN QUESTION (3/4 page)

Using the sample size formula for [test type]:
  NSN = 2 × [(z_α + z_β)² × σ²] / δ²

Parameters:
  • α = 0.05 (type I error, bilatéral) → z_α = 1.96
  • β = 0.10 (type II error) → z_β = 1.28 (power = 90%)
  • δ = [clinical difference from paper]
  • σ = [standard deviation from paper]
  
Calculation:
  NSN = 2 × [(1.96 + 1.28)² × σ²] / δ²
      = 2 × [10.49 × ...] / ...
      = [intermediate]
      
Adjustments:
  Dropout (X%): NSN / (1 - X/100) = ...
  Cluster (m=..., ICC=...): × [1 + (m-1) × ICC] = ...
  Final NSN: [round up]
